# Coupang 跨境电商智能选品系统 (Coupang Product Selection System)

## 项目简介

本项目是一个针对韩国电商平台 Coupang（酷澎）的自动化选品与数据分析系统。它通过自动化爬虫获取前台商品数据与后台运营数据，结合大语言模型（LLM）与 OCR 技术进行深度的图片与评论分析，最终提供多维度的市场趋势评估与利润测算，辅助业务团队做出科学的选品决策。

## 核心功能模块

本系统涵盖了需求文档中定义的七大核心步骤：
1. **数据爬取**：前台搜索抓取基础数据，后台匹配获取销量与点击量。
2. **数据筛选**：基于 30 天转化率与自定义规则过滤商品。
3. **深度分析**：发货方式识别、详情页视觉语义拆解、评论痛点与卖点挖掘。
4. **类目分析**：Naver 与 Coupang 趋势对比，GMV 预估与垄断程度分析。
5. **利润分析**：结合 1688 货源成本，自动核算利润率与 ROI。
6. **分析总结**：生成市场现状与机会报告。
7. **风险预警**：基于公司现状的选品风险评估。

## 环境搭建与运行指南

### 1. 环境要求
- Python 3.10+
- MySQL 8.0+
- 推荐使用虚拟环境 (venv/conda)

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 配置环境变量
复制 `.env.example` 文件并重命名为 `.env`，填入您的实际配置信息：
```bash
cp .env.example .env
```
请务必修改 `.env` 中的数据库密码（`DB_PASSWORD`）以及后续需要的 OpenAI API Key。

### 4. 初始化数据库
确保您的 MySQL 服务已启动，然后运行初始化脚本。该脚本会自动创建名为 `coupang_selection` 的数据库并建立所有必要的表结构：
```bash
python database/init_db.py
```

*(可选)* 如果需要导入测试数据以验证数据库是否正常工作，可执行：
```bash
mysql -u root -p coupang_selection < database/seeds/sample_data.sql
```

## 数据库设计概览

系统数据库采用关系型设计，核心表包括：
- `keywords`: 搜索关键词记录
- `categories`: 类目及市场趋势数据
- `products`: 产品基础信息
- `daily_metrics`: 每日点击量、销量、曝光量时序数据
- `detail_page_analysis`: 详情页视觉与语义分析结果 (JSON存储)
- `review_analysis`: 评论痛点与卖点提取汇总
- `profit_analysis`: 成本、运费与最终利润核算

详细的表结构与字段说明请参考 `database/schema.sql` 中的注释。
